Data-Driven Terrain Detection for Quadruped Locomotion

Approximated quadruped leg dynamics as a Koopman-based switched dynamic system, distinguishing swing and stance phases from encoder data alone — and predicting terrain stiffness from just a few seconds of stance data.

Motivation

Terrain-adaptive locomotion typically requires dedicated force/torque sensors or expensive exteroceptive sensing. This project asks: can we infer terrain stiffness from proprioceptive leg dynamics alone, using a data-driven approach?

Approach

  • Approximated the leg dynamics as a switched dynamic system — one linear system for swing phase, another for stance — using the Koopman operator framework.
  • Simulated different terrain stiffness behaviors (rigid, soft, variable) and collected leg state data across each terrain type.
  • Computed approximate Koopman generators for each terrain condition from the collected data using EDMD (Extended Dynamic Mode Decomposition).
  • Performed eigen analysis of the stance-phase Koopman generators — eigenvalue structure encodes terrain stiffness and can discriminate terrain type.
  • Demonstrated accurate terrain stiffness prediction using just a few seconds of stance data, with no external force sensors.
