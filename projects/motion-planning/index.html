Type: Course Project Period: Jan 2025 – May 2025 Course: Motion Planning — Prof. Nirav Patel, IIT Madras

Critical Comparison of Motion Planning Algorithms for a Mobile Manipulator with a 5-DOF Arm

Benchmarked A*, PRM, RRT, and RRT* on a KUKA YouBot (5-DOF arm + mobile base) in CoppeliaSim, with gradient-descent path smoothing applied to RRT output for real-world deployability.

Setup

The KUKA YouBot combines a differential-drive mobile base with a 5-DOF manipulator arm, requiring planning in both the base's 2D workspace and the arm's high-dimensional joint space. The simulation environment was built in CoppeliaSim with obstacle-cluttered scenarios.

Algorithms Compared

  • Mobile Base: A* (grid-based, optimal) vs PRM (probabilistic roadmap, sampling-based).
  • Arm (5-DOF joint space): RRT vs RRT* — comparing path quality, planning time, and smoothness.
  • Path Smoothing: Gradient descent post-processing applied to RRT-generated arm paths to eliminate excessive joint-space jitter and improve real-world deployability.

Key Findings

  • Decoupled planning (base and arm planned separately) outperformed joint planning in computation time, since A* solves the 2D base plan very quickly.
  • RRT* produced smoother arm paths than RRT at the cost of planning time; gradient-descent smoothing largely closed this gap for RRT.
  • Gradient-descent smoothing was essential for making RRT paths practically deployable on real hardware without excessive jerk.
