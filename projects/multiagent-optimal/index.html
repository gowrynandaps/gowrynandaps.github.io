Multi-Agent Planning Under Limited Communication via Game Theory & Optimal Control

Formulated decentralised multi-agent coordination as a differential game with Nash equilibrium, using Hamilton-Jacobi-Bellman equations over a directed communication graph — validated in MATLAB.

Problem Formulation

In multi-agent systems with limited communication bandwidth, agents cannot share full state information. This project models the coordination problem as a differential game where each agent optimises its own objective, leading to a Nash equilibrium solution that is stable without requiring global information.

Approach

  • Formulated the multi-agent system mathematically using differential games and Nash equilibrium-based cost functions.
  • Represented the communication topology as a directed graph, restricting each agent's information to its in-neighbours only.
  • Derived optimal control inputs using Hamilton-Jacobi-Bellman (HJB) equations, solved numerically for the coupled agent system.
  • Validated the model and control law successfully in a MATLAB numerical simulation, confirming convergence to Nash equilibrium.
