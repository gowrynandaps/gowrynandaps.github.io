Lyapunov-Based Control Barrier Functions for Safe 3D Navigation

A spatially-shifted CBF framework using a single Control Lyapunov Function and translational symmetry to synthesise multiple barrier functions online — validated on a 10-state quadrotor.

Key Contributions

  • Developed a spatially shifted CBF framework using a single CLF and translational symmetry to generate multiple barrier functions without offline computation.
  • Implemented an autonomous switching mechanism selecting the active CBF online based on obstacle proximity.
  • Real-time CBF-QP optimisation solving in under 30 ms on a 10-state quadrotor model.
  • Validated safety (no collision) and goal convergence in complex 3D simulation environments.
